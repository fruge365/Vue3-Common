# 🚀 Vue 3 + Vite 模板

> 📦 开箱即用的 Vue3 项目模板，克隆直接使用

## 📋 快速开始

### 📥 安装依赖
```bash
yarn
```

### 🏃‍♂️ 运行开发服务器
```bash
yarn dev
```

### 📦 构建生产版本
```bash
yarn build
```

## 🛠️ 技术栈

- **Vue 3** - 渐进式 JavaScript 框架
- **Vite** - 下一代前端构建工具
- **Pinia** - Vue 状态管理库
- **Naive UI** - Vue 3 组件库
- **JavaScript** - 编程语言
